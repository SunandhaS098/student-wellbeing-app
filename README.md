# Student Wellbeing Self-Assessment Tool

An interactive web application that analyzes student mental health trends and provides a personalized stress self-check tool.

## Features
- Data visualization of depression, anxiety, and CGPA trends
- Interactive stress self-assessment
- Risk categorization (Low / Moderate / High)
- Personalized recommendations
- Pie chart visualization of stress level

## Built With
- Python
- Streamlit
- Pandas
- Matplotlib

## How to Run

```bash
pip install streamlit pandas matplotlib
streamlit run app.py
